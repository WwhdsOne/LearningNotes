# 神经网络基础

神经网络（Neural Network）是深度学习的核心，其设计灵感来源于人脑中神经元之间的连接方式。下面围绕四个关键概念展开介绍。

---

## 1. 层次结构

神经网络由多层组成，每一层负责不同层次的特征提取：

| 层级 | 作用 | 说明 |
|------|------|------|
| **输入层** | 接收原始数据 | 如一张图片的像素值、一段文本的词向量 |
| **隐藏层** | 提取特征 | 一个或多个隐藏层，逐步抽象提取更高维特征 |
| **输出层** | 输出结果 | 如分类任务中的类别概率、回归任务中的预测值 |

> 层数越多，网络的表达能力越强，但训练难度也越大。

## 2. 神经元

神经元是神经网络的基本计算单元，模拟生物神经元的功能：

```
       x₁ ──→
              ┌─────────────┐
       x₂ ──→ │             │
              │  z = Σ(wᵢxᵢ) + b  │
       x₃ ──→ │             │
              │  a = f(z)   │
       xₙ ──→ │             │
              └──────┬──────┘
                     a（输出）
```

每个神经元执行两步计算：
1. **加权求和**：$z = \sum_{i=1}^{n} w_i x_i + b$
   - $x_i$：输入值
   - $w_i$：权重（Weight），表示该输入的重要性
   - $b$：偏置（Bias），调整激活阈值
2. **激活输出**：$a = f(z)$，通过激活函数 f 引入非线性

---

## 3. 全连接

全连接层（Fully Connected Layer / Dense Layer）是最基本的层结构：

- **定义**：前一层的每个神经元都与后一层的每个神经元相连
- **矩阵表示**：输出 = 权重矩阵 × 输入向量 + 偏置向量

$$
\begin{bmatrix} y_1 \\ y_2 \\ y_3 \end{bmatrix} = \begin{bmatrix} w_{11} & w_{12} \\ w_{21} & w_{22} \\ w_{31} & w_{32} \end{bmatrix} \times \begin{bmatrix} x_1 \\ x_2 \end{bmatrix} + \begin{bmatrix} b_1 \\ b_2 \\ b_3 \end{bmatrix}
$$

- **参数多**：假设输入有 1000 维，输出有 500 维，则参数量为 $1000 \times 500 + 500 = 500500$
- **用于末端**：现代网络中，全连接层通常放在网络末尾做最终分类或回归

## 4. 非线性

### 为什么需要非线性？

如果没有激活函数，无论多少层的神经网络都等价于单层线性变换，无法拟合复杂模式。

### 常用激活函数

| 激活函数 | 公式 | 特点 | 适用场景 |
|---------|------|------|---------|
| **Sigmoid** | $\sigma(x) = \frac{1}{1 + e^{-x}}$ | 输出 0~1，适合概率输出 | 二分类输出层 |
| **Tanh** | $\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$ | 输出 -1~1，以 0 为中心 | 隐藏层 |
| **ReLU** | $ReLU(x) = \max(0, x)$ | 计算快、缓解梯度消失 | **隐藏层最常用** |
| **Softmax** | $S(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$ | 多类别概率归一化 | 多分类输出层 |

### ReLU 的优势

- 计算简单，只需要比较和取最大值
- 在正区间梯度恒为 1，缓解梯度消失问题
- 引入稀疏性（负值区域输出为 0），使网络更高效

## 神经网络训练流程

```
1. 前向传播（Forward Propagation）
   数据从输入层 → 隐藏层 → 输出层，逐层计算

2. 计算损失（Loss Calculation）
   比较预测值与真实值的差距（如交叉熵损失、MSE）

3. 反向传播（Back Propagation）
   从输出层 → 输入层，逐层计算梯度

4. 参数更新（Parameter Update）
   用梯度下降或其变体（Adam、SGD等）更新权重和偏置

5. 重复 1-4，直到损失收敛
```

---

## 总结

| 概念 | 核心要点 |
|------|---------|
| 层次结构 | 输入层 → 隐藏层 → 输出层，逐层提取特征 |
| 神经元 | 加权求和 + 激活函数，是网络的基本计算单元 |
| 全连接 | 层间全连接，参数量大，多用于网络末端 |
| 非线性 | 通过激活函数引入，使网络能拟合复杂模式 |
