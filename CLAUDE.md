## Repository Overview

Personal learning notes repository (学习笔记), entirely in Chinese. Contains ~75 Markdown files across 16 topic directories covering Java backend, Go, DevOps, and interview prep. No source code, build system, or tests.

## Structure

- `Core Java/` — Java 核心技术笔记（卷一、卷二）
- `GO语言学习/` — Go 学习笔记，按库分类（fiber, gorm, casbin, 标准包）
- `面试/` — 面试准备材料，含面经和各专题总结
- `学成在线/` — 14 天课程项目笔记（最大目录）
- `chatgpt微服务应用/` — ChatGPT 微服务课程笔记
- `Docker/` `Git/` `linux/` — 运维工具笔记
- `工具安装/` — 工具安装和使用指南（含 Claude 使用指南）
- `DeepLearning/` `Javascript/` `IM应用/` `大创结题/` — 其他专题笔记
- `JAVA后端内容.md` — Java 后端学习路线图（含 B 站课程链接和进度追踪）
