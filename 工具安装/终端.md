# Zsh + Oh My Zsh + Starship (Catppuccin Powerline) 配置指南

## 1. 安装 Zsh

```bash
brew install zsh
```

将 Zsh 设为默认 Shell：

```bash
chsh -s $(which zsh)
```

## 2. 安装 Oh My Zsh

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

## 3. 安装 Oh My Zsh 插件

### zsh-autosuggestions（命令自动建议）

```bash
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
```

### zsh-syntax-highlighting（语法高亮）

```bash
git clone https://github.com/zsh-users/zsh-syntax-highlighting ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
```

### zsh-autocomplete（前缀补全）

```bash
git clone --depth 1 https://github.com/marlonrichert/zsh-autocomplete ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autocomplete
```

在 `~/.zshrc` 中启用插件（找到 `plugins=(...)` 行，修改为）：

```bash
plugins=(
    git
    zsh-autosuggestions
    zsh-syntax-highlighting
    zsh-autocomplete
)
```

## 4. 安装 Starship

```bash
brew install starship
```

在 `~/.zshrc` 末尾添加（必须在 `source $ZSH/oh-my-zsh.sh` 之后）：

```bash
eval "$(starship init zsh)"
```

## 5. 安装 Nerd Font

```bash
brew tap homebrew/cask-fonts
brew install --cask font-hack-nerd-font
```

然后在终端偏好设置中将字体改为 **Hack Nerd Font**。

## 6. 配置 Catppuccin Powerline 主题

使用 Starship 官方预设一键配置：

```bash
starship preset catppuccin-powerline -o ~/.config/starship.toml
```

默认使用 Mocha 风格，可通过修改 `~/./config/starship.toml` 中的 `palette` 值切换：

- `catppuccin_mocha`（默认，深色）
- `catppuccin_frappe`
- `catppuccin_macchiato`
- `catppuccin_latte`（浅色）

## 7. 生效

```bash
source ~/.zshrc
```
