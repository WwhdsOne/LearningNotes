# Zsh + Starship + Catppuccin + Powerline 配置指南

## 1. 安装 Starship

```bash
brew install starship
```

在 `~/.zshrc` 末尾添加：

```bash
eval "$(starship init zsh)"
```

## 2. 安装 Powerline 字体

```bash
brew tap homebrew/cask-fonts
brew install --cask font-hack-nerd-font
```

然后在终端偏好设置中将字体改为 **Hack Nerd Font**。

## 3. 配置 Catppuccin 主题

创建 Starship 配置文件：

```bash
mkdir -p ~/.config && touch ~/.config/starship.toml
```

写入以下内容（Catppuccin Mocha 风格）：

```toml
format = """
$directory\
$git_branch\
$git_status\
$nodejs\
$python\
$golang\
$rust\
$cmd_duration\
$line_break\
$character"""

[character]
success_symbol = "[❯](bold green)"
error_symbol = "[❯](bold red)"

[directory]
style = "bold mauve"
truncation_length = 3
truncate_to_repo = true

[git_branch]
style = "bold pink"
symbol = " "

[git_status]
style = "bold red"

[nodejs]
style = "bold green"
symbol = " "

[python]
style = "bold yellow"
symbol = " "

[golang]
style = "bold cyan"
symbol = " "

[rust]
style = "bold peach"
symbol = " "

[cmd_duration]
style = "bold yellow"
min_time = 2_000
```

## 4. 前缀补全（基于 zsh-autocomplete）

```bash
# 安装 zsh-autocomplete
brew install zsh-autocomplete
```

在 `~/.zshrc` 中添加（放在 `eval "$(starship init zsh)"` 之前）：

```bash
source $(brew --prefix)/share/zsh-autocomplete/zsh-autocomplete.plugin.zsh
```

## 5. 生效

```bash
source ~/.zshrc
```

## 备选：手动安装 zsh-autocomplete

如果 brew 未收录，可用 git 方式：

```bash
git clone --depth 1 https://github.com/marlonrichert/zsh-autocomplete ~/.zsh/plugins/zsh-autocomplete
```

然后在 `~/.zshrc` 中：

```bash
source ~/.zsh/plugins/zsh-autocomplete/zsh-autocomplete.plugin.zsh
```
