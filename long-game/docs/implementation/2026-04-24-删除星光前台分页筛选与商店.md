## Redis 迁移

默认 `redis_prefix` 为 `vote:button:` 时使用：

```sh
BUTTON_PREFIX='vote:button:'
NS='vote:'

redis-cli DEL "${NS}buttons:starlight"

redis-cli --raw --scan --pattern "${BUTTON_PREFIX}*" | while IFS= read -r key; do
  redis-cli HDEL "$key" starlight_eligible
done

redis-cli --raw --scan --pattern "${NS}user-cosmetics:*" | while IFS= read -r key; do
  redis-cli DEL "$key"
done

redis-cli --raw --scan --pattern "${NS}user-cosmetic-loadout:*" | while IFS= read -r key; do
  redis-cli DEL "$key"
done
```

如果线上 `redis_prefix` 不是 `vote:button:`，先按后端规则推导命名空间：去掉末尾 `button:` 后作为 `NS`，完整按钮前缀作为 `BUTTON_PREFIX`。
