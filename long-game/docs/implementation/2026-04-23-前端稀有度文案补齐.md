# 2026-04-23 前端稀有度文案补齐

## 背景

前端部分装备名称通过颜色表达稀有度，但没有在所有展示位置同步写出明确的稀有度文字，识别成本较高。

## 本次调整

- 新增统一文案工具 `formatRarityLabel(value)`，输出 `稀有度：X`
- 公共页所有彩色装备名展示旁补充明确稀有度文案
- 外观商店稀有度展示改为统一文案格式
- 管理端装备列表补充统一稀有度文案

## 验证

- `npm test -- src/utils/rarity.test.js src/pages/PublicPage.rarityDisplay.test.js src/components/admin/AdminEquipmentTab.rarityDisplay.test.js`
- `npm run build`
