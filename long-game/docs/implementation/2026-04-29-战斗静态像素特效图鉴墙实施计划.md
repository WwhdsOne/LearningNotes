# 战斗静态像素特效图鉴墙实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 按任务逐项执行。步骤使用复选框语法记录进度。

**Goal:** 新增一个内部 Demo 页面，使用静态像素图方式展示当前 BattlePage 已接线的战斗特效，并验证多特效并发时的分层与可读性。

**Architecture:** 仅改前端，不接后端，不改 BattlePage 真实触发链路。新增独立 Demo 页面组件，复用现有 Boss 格子视觉语言，在页面内部构造静态图鉴墙与组合预览区。所有战斗特效预览直接使用本地静态像素构图，不走 OSS 映射。

**Tech Stack:** Vue 3、Vite、现有全局 `style.css`、项目内隐藏 Demo 路由分支、Vitest 静态源码测试
