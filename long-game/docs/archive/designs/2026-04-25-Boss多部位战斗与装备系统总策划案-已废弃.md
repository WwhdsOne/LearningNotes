## 2. 核心玩法定位

玩家点击 Boss 的不同部位造成伤害。每个部位拥有独立血量、护甲和类型；所有部位被击碎后 Boss 死亡，系统进入下一关或结算掉落。

长期循环：

1. 选择或继承当前主系。
2. 进入 Boss 战，按部位类型选择攻击目标。
3. 击碎部位并触发主系技能效果。
4. Boss 死亡后获得 3 至 6 件装备。
5. 通过装备、强化、突破、品质和技能树继续提升战斗效率。

核心特色：

| 特色 | 说明 |
| --- | --- |
| 多部位 Boss | 5x5 坐标布局支持 3 至 25 个部位 |
| 三流派克制 | 普攻克软组织，破甲克重甲，暴击克弱点 |
| 减法防御 | 护甲直接扣减伤害，数值透明 |
| 自动接替 | 页面失焦或长时间无操作后自动战斗接手 |
| 装备联动 | 掉落按主系 60%、副系合计 30%、通用 10% 分配 |
| 前端反馈 | 部位、伤害类型、技能触发均有独立视觉反馈 |

## 3. Boss 部位规则

### 3.1 部位数量

| Boss 阶段 | 部位数量 | 掉落件数 | 最高稀有度 |
| --- | --- | --- | --- |
| 新手 Boss | 3 个 | 3 件 | 稀有 |
| 常规 Boss | 5 至 7 个 | 4 件 | 史诗 |
| 高阶 Boss | 7 至 9 个 | 5 件 | 传说 |
| Raid Boss | 9 至 15 个 | 6 件 | 神话 |
| 终极 Boss | 15 至 25 个 | 6 件 | 至臻 |

### 3.2 5x5 坐标空间

- 坐标范围固定为 `(0,0)` 至 `(4,4)`，左上角为 `(0,0)`。
- 每格支持三种状态：空位、常规部位、大型部位。
- 常规部位占 1 格，直接配置类型、血量和护甲。
- 大型部位预留为 2x2 或更大范围，当前版本只要求数据结构可扩展。
- 前端按 `layout[]` 动态渲染，不写死按钮数量和位置。

### 3.3 血量体系

- Boss 总血量 = 所有存活部位当前血量之和。
- 每个部位显示独立小血条。
- 部位血量归零后变灰、下陷并禁用点击。
- 所有部位血量归零后 Boss 死亡并进入结算。

### 3.4 部位类型

| 类型 | 标识色 | 护甲 | 伤害系数 | 克制流派 | 默认视觉 |
| --- | --- | --- | --- | --- | --- |
| 软组织 | 绿 | 低 | 1.0 | 普攻流 | 绿色边框微光 |
| 重甲 | 灰 | 高 | 0.4 | 破甲流 | 灰色金属边框 |
| 弱点 | 红 | 中 | 2.5 | 暴击流 | 红色脉动光晕 |

## 4. 伤害公式

基础公式：

```text
最终伤害 = max(攻击力 * 部位伤害系数 - 目标有效护甲, 1)
  * (1 + 全伤害增幅 + Boss 增伤)
  * 暴击乘区
```

参数规则：

| 参数 | 规则 |
| --- | --- |
| 攻击力 | 来自装备实例、品质、强化、突破和技能效果 |
| 目标有效护甲 | 原始护甲 * (1 - 破甲率)，破甲率上限 80% |
| 全伤害增幅 | 装备、词缀、技能树的同类加法叠加 |
| Boss 增伤 | 对 Boss 的独立增伤，和全伤害增幅加法叠加 |
| 暴击乘区 | 未暴击为 1.0，暴击时使用暴击伤害倍率 |
| 最低伤害 | 最低为 1，避免无法破防 |

额外伤害：

- 追击伤害：普攻流专属，按技能规则追加，忽略护甲。
- 真实伤害：破甲粉碎、暴击出血等效果直接扣血，不进入减法护甲公式。
- 最大生命削减：审判日、终末血斩等按部位最大生命或 Boss 总生命结算，并同步刷新总血条。

## 5. 三主系技能树

通用规则：

- 玩家选择一个主系后按层级向下点亮。
- 可选择另一个系作为副系，但副系只能点非基石、非终极的 2 个节点。
- 所有影响攻击速度、点击频率、点击间隔、自动额外点击概率的节点永久禁止。
- 配置字段以 `talentId`、`tree`、`tier`、`effectType`、`effectValue`、`vfxId`、`prerequisite` 为核心。

### 5.1 普攻：均衡攻势

定位：稳定均衡，越打越痛，挂机容错高。

| 层级 | 名称 | 效果 | 前端特效 |
| --- | --- | --- | --- |
| 基石 | 暴风连击 | 对同一部位累计攻击 200 次后，接下来 15 次点击每次附加攻击力 50% 的无视护甲追击伤害 | 屏幕边缘绿色光晕，点击时额外绿色刀光，追击数字以蓝色显示 |
| 二层 A | 攻击强化 | 攻击力 +25% | 常驻被动 |
| 二层 B | 伤害增幅 | 全伤害增幅 +20% | 常驻被动 |
| 二层 C | 软组织特攻 | 对软组织部位伤害 +40% | 命中软组织时绿色边框短暂高亮 |
| 三层 A | 蓄力返还 | 暴风连击触发后保留该部位 30% 累计次数 | 弹出“蓄力返还”并播放绿色环回收动画 |
| 三层 B | 追击强化 | 追击伤害提升至攻击力 80% | 追击数字放大，刀光变粗 |
| 三层 C | 连击扩展 | 暴风连击持续次数提升至 25 次 | 光晕持续时间延长 |
| 四层 A | 围剿 | Boss 每存活一个部位，所有伤害 +12% | 技能栏旁显示存活部位计数 |
| 四层 B | 残血收割 | 部位血量低于 25% 时，对其伤害翻倍 | 低血量部位出现橙色断裂光效 |
| 终极 | 白银风暴 | 击碎任意部位立即触发 15 击版暴风连击；持续期间所有部位视为软组织 | 全屏银色闪烁，光标变银，所有部位绿色高亮 |

### 5.2 破甲：碎盾攻坚

定位：重甲克星，适合高护甲 Boss 和越级攻坚。

| 层级 | 名称 | 效果 | 前端特效 |
| --- | --- | --- | --- |
| 基石 | 灭绝穿甲 | 永久无视 40% 护甲；对重甲累计 200 次触发结构崩塌，使目标护甲归零 8 秒 | 攻击时出现裂纹光效；崩塌时部位碎裂高亮，护甲图标碎裂 |
| 二层 A | 穿甲强化 | 破甲率额外 +25% | 常驻被动 |
| 二层 B | 首领猎杀 | Boss 增伤 +30% | 对 Boss 出现微弱十字准星 |
| 二层 C | 以强制强 | 攻击重甲时，目标每有 100 护甲，伤害 +2% | 数字旁显示护甲图标 |
| 三层 A | 重甲特攻 | 攻击重甲部位伤害 +50% | 命中重甲时出现锤击裂地特效 |
| 三层 B | 崩塌延长 | 结构崩塌持续时间提升至 15 秒 | 黑闪状态延长 |
| 三层 C | 自动打击 | 每 20 秒自动对血量最高重甲造成 300% 攻击力必中打击 | 灰色重锤从上方砸向目标 |
| 四层 A | 废墟打击 | 崩塌状态下目标所受伤害再 +100% | 崩塌部位叠加灰暗裂纹滤镜 |
| 四层 B | 破甲转化 | 破甲率的 50% 转为全伤害增幅 | 技能栏旁显示当前转化值 |
| 终极 | 审判日 | 对同一重甲累计 500 次，直接削除 50% 最大生命值，每场战斗仅一次 | 目标爆炸，全屏震动 0.3 秒，巨大紫色数字弹出 |

### 5.3 暴击：致命洞察

定位：极限爆发，通过暴击、死兆和终结标记形成高峰值输出。

| 层级 | 名称 | 效果 | 前端特效 |
| --- | --- | --- | --- |
| 基石 | 溢杀 | 基础暴击率 +20%；暴击率超过 100% 的部分每 1% 转为 +2% 暴击伤害；对弱点暴击获得 1 层死兆 | 暴击数字金色；弱点暴击金色红描边；死兆层数显示为暗红环 |
| 二层 A | 死兆共鸣 | 每层死兆使暴击伤害倍率额外 +0.3% | 死兆环随层数加深 |
| 二层 B | 残忍 | 暴击伤害倍率 +60% | 暴击数字进一步放大 |
| 二层 C | 剥皮 | 对非弱点暴击时有 30% 概率使其 5 秒内视为弱点 | 部位红色闪烁 3 次并显示弱点虚影 |
| 三层 A | 致命出血 | 暴击必附出血，4 秒内流失该次暴伤 60% 的真实伤害，多层独立 | 滴血粒子和独立紫色真实伤害数字 |
| 三层 B | 斩杀预兆 | 部位血量低于 35% 时，每层死兆使其受伤害 +1% | 低血量部位出现暗红雾气 |
| 三层 C | 死兆收割 | 每累计 30 层死兆，发动一次必暴伤害，不消耗层数 | 血色镰刀划过目标 |
| 四层 A | 死亡狂喜 | 消耗 50 层死兆，6 秒内暴伤 +200%，且全攻击视为弱点 | 屏幕边缘红光脉冲，死兆环燃烧旋转 |
| 四层 B | 终末血斩 | 每 120 次暴击斩去部位 12% 最大生命值，30 秒最多一次 | 血色巨刃下劈，冷却环显示倒计时 |
| 终极 | 末日审判 | 开局标记 2 个终结部位；击碎标记获得 100 层死兆和下次暴伤 x3；持有强化再击碎另一标记则 x6 | 红色准星悬浮；击碎时全屏血色闪烁，暴击数字极限放大 |

## 6. 装备与掉落联动摘要

装备完整规则见 [装备稀有度与成长数值规范](../specs/2026-04-23-装备稀有度与成长数值规范.md)。Boss 战斗侧只依赖以下结论：

- 每次 Boss 击杀掉落 3 至 6 件装备。
- 至少 1 件为该 Boss 阶段的最高掉落稀有度；终极 Boss 的至臻保底按装备规范执行。
- 每件装备独立按主系 60%、副系合计 30%、通用 10% 决定掉落池。
- 稀有度固定为普通、稀有、史诗、传说、神话、至臻。
- 装备实例使用 `instanceId`、`prototypeId`、`enhanceLevel`、`breakthroughLevel`、`affixes`、`baseStats`、`quality`、`bindStatus`、`obtainSource` 等字段。
- 禁止出现影响攻击速度、点击间隔、额外点击概率的词缀。

主系专精掉落：

| 主系 | 偏好部位 | 代表词缀方向 |
| --- | --- | --- |
| 普攻流 | 软组织 | 攻击力、追击、连击增伤、稳定全伤 |
| 破甲流 | 重甲 | 破甲率、护甲归零、Boss 增伤、重甲真伤 |
| 暴击流 | 弱点 | 暴击率、暴击伤害、溢暴转化、弱点暴击 |

## 7. 前端反馈规范

### 7.1 伤害数字

| 类型 | 颜色 | 大小 | 特效 |
| --- | --- | --- | --- |
| 普通伤害 | 白色 | 基准 | 向上飘散 |
| 暴击伤害 | 金色 | x1.5 | 轻微震动 0.1 秒 |
| 弱点暴击 | 金色红描边 | x2.0 | 震动 0.15 秒 |
| 追击伤害 | 蓝色 | 基准 | 与主伤害错开弹出 |
| 真实伤害 | 紫色 | 基准 | 闪电图标前缀 |
| 最大生命削减 | 巨大紫色 | x2.5 至 x3.0 | 全屏震动或巨刃特效 |
| 终极暴击 | 金色极限 | x4.0 | 慢镜头和暗角 |

同帧多数字必须自动错开位置。优先级为：末日审判、审判日、弱点暴击、暴击、真实伤害、追击、普通伤害。

### 7.2 部位与状态反馈

| 状态 | 表现 |
| --- | --- |
| 可攻击 | 按部位类型显示绿、灰、红边框 |
| 被击碎 | 变灰、下陷、禁用点击 |
| 结构崩塌 | 灰黑闪烁，裂纹叠加，护甲图标碎裂 |
| 出血 | 滴血粒子和血滴层数图标 |
| 剥皮 | 红色闪烁后短暂视为弱点 |
| 终结标记 | 红色准星悬浮旋转 |

Boss 总血条下方展示 Debuff 图标区，至少支持护甲归零、出血、终结标记、剥皮、审判日已触发。

### 7.3 性能降级

所有特效必须可通过设置降级：

- 低性能模式关闭屏幕震动、全屏闪烁和高密度粒子。
- 伤害数字保留，但减少同时存在数量。
- 状态识别优先于装饰效果，不能因降级导致玩家无法判断部位类型和技能状态。

## 8. 扩展预留

基于 5x5 坐标的后续扩展：

| 类型 | 逻辑 |
| --- | --- |
| 溅射 | 目标相邻 1 至 2 格部位受到百分比伤害 |
| 穿透 | 目标所在行或列受到伤害 |
| 连锁 | 向最近存活部位跳跃，每次衰减 |
| 领域 | 指定坐标为中心，对 3x3 范围持续伤害 |

Boss 机制预留：

- 部位再生：击碎后 N 秒在空位重生。
- 护盾部位：必须先击破护盾才能伤害关键部位。
- 相邻联动：部位存亡影响相邻部位属性。
- 世界 Boss：多玩家分工锁定部位，统计部位击碎贡献。
