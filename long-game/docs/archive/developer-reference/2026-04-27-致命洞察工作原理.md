## 核心机制

致命洞察围绕“暴击 -> 死兆堆叠 -> 被动档位增伤 / 低血加成 / 终结技”展开。

当前后端口径：

- `crit_core` 提供基础暴击率、溢出暴击转暴伤、死兆获取入口
- `crit_omen_reap` 已改为被动档位增伤，不再消耗死兆
- `crit_doom_judgment` 已改为 Tier1 小技能，不再是旧终极
- `crit_death_ecstasy` 已改为 Tier4 终极，按等级提升暴伤倍率

## 当前节点口径

### 主节点

| ID | 名称 | 层级 | 当前效果摘要 |
|---|---|---|---|
| `crit_core` | 溢杀 | Tier0 基石 | 基础暴击率成长、溢出暴击转暴伤、弱点暴击额外死兆 |
| `crit_omen_resonate` | 死兆共鸣 | Tier1 | 每层死兆暴伤成长 |
| `crit_cruel` | 残忍 | Tier1 | 暴击伤害成长型加成 |
| `crit_skinner` | 剥皮 | Tier1 | 强制视为弱点的概率与持续时间成长 |
| `crit_doom_judgment` | 末日审判 | Tier1 | 标记数与每标记死兆奖励成长 |
| `crit_bleed` | 致命出血 | Tier2 | 暴击附加真伤比例成长 |
| `crit_omen_kill` | 斩杀预兆 | Tier2 | 低血阈值与每层死兆额外增伤成长 |
| `crit_omen_reap` | 死兆收割 | Tier2 | 30/60/90/120 层时自动提升档位伤害，不消耗层数 |
| `crit_final_cut` | 终末血斩 | Tier3 | 所需暴击次数下降、削除比例提升 |
| `crit_death_ecstasy` | 死亡狂喜 | Tier4 | 100 层死兆触发终结伤害，暴伤倍率随等级成长 |

## 层满奖励

| 层级 | 当前奖励 |
|---|---|
| Tier0 | 全伤害 +10% |
| Tier1 | 暴击率 +10% |
| Tier2 | 斩杀血线 +5% |
| Tier3 | 每层死兆暴伤 +0.5% |
| Tier4 | 狂喜倍率 +2x |

## 成本口径

致命洞察与其他两系共用统一成本模型：

```text
cost(level) = round(baseCost × level^0.85 × 1.8)
```

基准成本：

- Tier0 基石：`20`
- Tier1 主节点：`30`
- Tier2 主节点：`80`
- Tier3 主节点：`150`
- Tier4 终极：`200`
- Tier1 小节点：`15`
- Tier2 小节点：`35`
- Tier3 小节点：`60`

## 已废弃口径

以下说法均已废弃：

- “死兆收割消耗 30 层”的旧机制
- “死亡狂喜 50 层触发 6 秒 buff”的旧机制
- “末日审判是 Tier4 终极”的旧结构
- 固定成本 `100 / 2000 / 8000 / 30000 / 120000`
