## 一、核心规则

| 规则             | 说明                                                   |
| ---------------- | ------------------------------------------------------ |
| 点击无金币       | 手动点击Boss部位不产生金币                             |
| 仅Boss击杀有金币 | 所有金币来自Boss被击杀的瞬间                           |
| 挂机无实时收益   | 离开页面期间不累积任何即时资源                         |
| 挂机伤害×0.5     | 自动战斗每次伤害为手动伤害的一半                       |
| 挂机无特效       | 自动战斗不触发暴击、不触发天赋爆发、不触发任何概率效果 |
| Boss后台自动切换 | 所有玩家同时面对同一个Boss，Boss由后台按规则自动切换   |

## 五、挂机系统

### 5.1 挂机触发

服务器有一个定时器，每秒扫描一次所有玩家状态。如果玩家处于挂机状态（页面失焦或无操作超过设定时长），则执行一次自动战斗。

### 5.2 挂机执行流程

定时器每秒对每个挂机玩家执行：

1. 获取服务器当前Boss
2. 如果Boss已切换且该玩家未参与过当前Boss，将玩家状态同步到新Boss初始状态
3. 获取当前Boss所有存活部位列表
4. 从存活部位中随机选择一个作为目标
5. 对该部位造成一次伤害，伤害值 = 玩家当前攻击力 × 0.5
6. 如果该部位血量归零，标记为击碎
7. 如果所有部位全部击碎，Boss死亡，玩家获得挂机击杀奖励，进入下一个Boss状态

### 5.3 挂机规则

| 规则     | 说明                                                     |
| -------- | -------------------------------------------------------- |
| 执行频率 | 每秒一次                                                 |
| 目标选择 | 从当前Boss存活部位中随机选一个                           |
| 伤害数值 | 玩家攻击力 × 0.5                                         |
| 特效     | 不触发暴击、不触发任何天赋效果、不触发任何概率事件       |
| Boss切换 | 切换后挂机玩家自动同步到新Boss，之前未打完的部位进度作废 |

### 5.4 回归弹窗

玩家从挂机状态回归时，弹出结算弹窗展示挂机期间击杀了多少个Boss、获得的金币和强化石总数。

## 六、Boss配置字段

每个Boss在配置表中新增以下字段：

| 字段        | 类型 | 说明               |
| ----------- | ---- | ------------------ |
| goldOnKill  | int  | 手动击杀奖励金币   |

## 七、数值验证

### 7.1 手动玩家

| 阶段   | 单次Boss金币 | 单次Boss强化石 | 典型强化消耗                            | 需要击杀次数           |
| ------ | ------------ | -------------- | --------------------------------------- | ---------------------- |
| 新手期 | 500          | 1              | 普通+0→+5 累计1,200金币+10强化石        | 金币3次，强化石10次    |
| 成长期 | 8,000        | 10             | 史诗+10→+11 本级20,000金币+70强化石     | 金币3次，强化石7次     |
| 后期   | 100,000      | 80             | 至臻+29→+30 本级1268万金币+22,803强化石 | 金币127次，强化石285次 |

### 7.2 挂机玩家

挂机伤害×0.5，挂机击杀金币×0.5，挂机击杀强化石×0.5后向下取整。

挂机金币效率约为手动的25%（击杀速度减半 × 金币减半）。

| 阶段   | 挂机单次Boss金币 | 挂机单次Boss强化石 |
| ------ | ---------------- | ------------------ |
| 新手期 | 250              | 0                  |
| 成长期 | 4,000            | 5                  |
| 后期   | 50,000           | 40                 |
