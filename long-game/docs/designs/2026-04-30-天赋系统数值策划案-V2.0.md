## 一、全局数值规则

### 1.1 乘区体系

| 乘区 | 包含效果类型 | 叠加规则 |
|------|------------|---------|
| 攻击力区 | `attack_power_percent` | 同类加算 |
| 全伤害区 | `all_damage_amplify` | 同类加算 |
| 部位/类型增伤区 | `part_type_damage`（软组织/重甲） | 同类加算 |
| 条件增伤区 | `per_part_damage`、`low_hp_bonus`、`armor_scaling`、`pen_to_amplify`、`omen_low_hp` | 各自独立，触发时乘算 |
| 暴击伤害区 | `crit_damage_bonus`、`omen_per_stack_crit_dmg` | 同类加算 |
| 弱点暴击区 | `weakspot_insight` | 独立乘区 |
| 追击倍率区 | `chase_upgrade` | 独立乘区 |
| 崩塌伤害区 | `collapse_damage_amp` | 同类取最高 |
| 死兆收割区 | `omen_reap_passive` | 独立档位乘区 |
| 终结技能区 | `silver_storm`、`judgment_day`、`final_cut` | 各自独立乘区 |
| 附加伤害区 | `bleed` | 独立乘区 |

### 1.2 死兆机制

| 参数 | 数值 |
|------|------|
| 上限 | **150 层** |
| 获取途径 | ① 弱点暴击 +1/次（溢杀 T0）② 击碎末日审判标记部位 +15~27/个 |
| 消耗出口 | **仅有终末血斩（T4），150 层时自动触发，消耗全部** |
| 满层行为 | 立即触发终末血斩，层数归零 |
| 中途消耗 | 无。死兆只增不减，不存在 50 层手动释放的中途出口 |

### 1.3 条件触发类技能规则

所有带持续时间的被动触发技能统一采用 **“触发后进入冷却”** 机制。

| 技能 | 触发条件 | Lv5 持续时间 | Lv5 冷却时间 | 覆盖率 |
|------|---------|------------|------------|--------|
| 残血收割（普攻 T3） | Boss HP ≤ 阈值 | 7s | 18s | 28% |
| 崩塌共振（破甲 T2） | 重甲部位崩塌 | 随崩塌窗口（6s） | 10s | 受崩塌频率限制 |
| 剥皮（暴击 T1） | 暴击命中 | 10s | 20s | 33% |

### 1.4 数值校准基准

- 满配玩家对克制目标，**爆发期 50~100 倍基础伤害**，**非爆发期 10~20 倍**
- 暴击系对弱点目标峰值约 ×960（平A）/ ×2880（血斩）；对非弱点约 ×274
- Boss 战期望时长：未点天赋 120s → 满配天赋 15~25s
- 三系在各自优势场景下伤害差距控制在 ±15% 以内

## 四、暴击系：致命洞察

**定位：** 弱点特化 + 死兆循环  
**核心资源：** 死兆层数（上限 150，只增不减，满层自动触发血斩）  
**适配场景：** 弱点明显的 Boss  

### 4.1 玩法闭环

```text
弱点暴击 → 死兆 +1
     ↓
死兆累积 → 收割档位逐步解锁（1.10 → 1.50）
     ↓
常驻生效：弱点洞察（弱点暴击放大）
         致命出血（暴击附加流血）
         剥皮（CD 制临时弱点）
         斩杀预兆（低血量每死兆增伤）
     ↓
死兆 150 → 自动清空 + 终末血斩
     ↓
归零 → 重新攒
```

### 4.5 满配伤害乘算（策划目标值口径）

#### 巅峰状态（150 死兆、斩杀预兆生效、打弱点）

|------|------|------|------|

## 八、开发实现注意事项

### 8.1 乘区计算顺序

```text
最终伤害 = 基础攻击力
  × (1 + 攻击力加成)                                    ← 攻击力区
  × (1 + 全伤害加成)                                    ← 全伤害区
  × (1 + 部位类型加成)                                  ← 部位区
  × Π(条件增伤)                                         ← 条件增伤乘积
  × (1 + 暴击伤害)                                      ← 暴伤区（若暴击）
  × 弱点洞察倍率                                        ← 弱点区（若目标为弱点）
  × 追击倍率                                            ← 追击区
  × 崩塌加成                                            ← 崩塌区（若崩塌中）
  × 死兆收割档位                                        ← 收割区（若达标）
  × 终结技能倍率                                        ← 终结区（若触发）
  × 附加伤害倍率                                        ← 附加区（若触发）
```

### 8.2 冷却冲突处理

- 残血收割、崩塌共振、剥皮各自独立计时
- 冷却期间再次满足触发条件 → 不触发，等待冷却结束
- 冷却结束且条件仍满足 → 立即触发

### 8.3 死兆系统事件

- `ON_OMEN_CHANGE`：层数变动，UI 更新
- `ON_OMEN_FULL`：达到 150 层，触发终末血斩
- `ON_OMEN_RESET`：血斩后归零，重置收割档位
- 达到 150 层后，获取死兆的途径不再生效（溢出保护）

### 8.4 剥皮触发条件

- 检查：①不在 CD 中；②存在非弱点部位
- 两者有一不满足则不触发（不消耗 CD）

### 8.5 末日审判触发保护

- 每场 Boss 战仅触发一次
- 触发后标记 `has_triggered_doom = true`，Boss 战结束后重置
