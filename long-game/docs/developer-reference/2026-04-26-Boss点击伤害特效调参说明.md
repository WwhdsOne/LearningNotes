# Boss 点击伤害特效调参说明

本文说明本次“Boss 点击伤害数字特效”实装后的可调参数位置与调整方式。

## 1. 入口文件

1. 前端状态与特效判定：`frontend/src/pages/publicPageState.js`
2. 页面渲染层：`frontend/src/pages/BattlePage.vue`
3. 动画与视觉样式：`frontend/src/style.css`

## 2. 当前已实装效果

1. 伤害类型表现：普通、暴击、弱点暴击、追击、真实伤害、末日审判、审判日。
2. 同帧多数字错位：同一个部位在短时间连续弹字，会自动按预设轨迹错位，避免重叠。
3. 大范围弹字：数字可在按钮外侧出现，不再限制在按钮内部。
4. 舞台特效：震动、闪光、暗角、慢镜头、巨刃扫过。
5. 纯数字展示：弹字仅显示数值，不显示“伤害类型文字”。

## 3. 如何调整“大小、时长、优先级、触发逻辑”

### 3.1 调整各类型大小和持续时间

文件：`frontend/src/pages/publicPageState.js`

常量：`DAMAGE_VARIANTS`

- `scale`：数字整体倍率
- `ttl`：弹字存在时长（毫秒）
- `shake`：该类型触发的震动时长（毫秒）
- `stageFx`：该类型触发的舞台特效列表

示例：

```js
judgement: {
  scale: 4,
  ttl: 2200,
  shake: 180,
  stageFx: ['shake', 'slowMo', 'vignette'],
}
```

### 3.2 调整优先级

文件：`frontend/src/pages/publicPageState.js`

常量：`DAMAGE_PRIORITY`

当前顺序（高到低）：

1. `doomsday`（末日审判）
2. `judgement`（审判日）
3. `weakCritical`（弱点暴击）
4. `critical`（暴击）
5. `trueDamage`（真实伤害）
6. `pursuit`（追击）
7. `normal`（普通）

直接调整数组顺序即可改变冲突时的显示优先级。

### 3.3 调整触发阈值（大招触发频率）

文件：`frontend/src/pages/publicPageState.js`

常量：

- `DAMAGE_JUDGEMENT_THRESHOLD_RATIO`
- `DAMAGE_DOOMSDAY_THRESHOLD_RATIO`

含义：单次伤害占 Boss 最大生命比例达到阈值后，自动升级为对应高阶特效。

### 3.4 调整同帧错位的散开幅度

文件：`frontend/src/pages/publicPageState.js`

函数：`nextBurstOffset(key, variant)`

可调项：

1. `pattern`：首批弹字坐标模板（控制左右与上下分布）
2. `shift`：溢出到下一层时的偏移增量
3. `variantBias`：高阶特效额外抬高量

## 4. 如何调整“视觉风格与动画”

文件：`frontend/src/style.css`

### 4.1 数字视觉（颜色、字号、描边、光效）

- 基础类：`.boss-zone-button__damage-burst`
- 各类型类：
  - `.boss-zone-button__damage-burst--normal`
  - `.boss-zone-button__damage-burst--critical`
  - `.boss-zone-button__damage-burst--weakCritical`
  - `.boss-zone-button__damage-burst--pursuit`
  - `.boss-zone-button__damage-burst--trueDamage`
  - `.boss-zone-button__damage-burst--doomsday`
  - `.boss-zone-button__damage-burst--judgement`

### 4.2 动画节奏

关键帧：

1. `@keyframes boss-damage-float`：主漂浮动画
2. `@keyframes boss-damage-shake`：数字局部抖动
3. `@keyframes boss-stage-shake`：舞台震动
4. `@keyframes boss-stage-flash`：闪光
5. `@keyframes boss-stage-doom`：末日脉冲
6. `@keyframes boss-stage-blade`：巨刃扫过
7. `@keyframes boss-stage-vignette`：暗角
8. `@keyframes boss-stage-slowmo`：慢镜头

### 4.3 是否允许弹字超出按钮

- 当前已开启：`.boss-zone-button { overflow: visible; }`
- 如需限制回按钮内：改回 `overflow: hidden;`

## 5. 后端字段扩展建议（可选）

当前前端已经兼容以下可选字段：

- `payload.damageType` / `payload.hitType` / `payload.effectType`
- `payload.trueDamage`
- `payload.ignoreArmor`
- `payload.pursuit`
- `payload.maxHpCut`
- `payload.doomsday`
- `payload.ultimateCritical`
- `payload.judgementDay`
- `payload.bossDamage`

后端如果补齐这些字段，前端会优先按显式字段判定，不再依赖兜底推断。
