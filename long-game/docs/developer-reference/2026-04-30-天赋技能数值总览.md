## 触发型效果速览

| 天赋 | 触发条件 | 效果 |
| --- | --- | --- |
| 暴风连击 | 点击次数达标（30~50） | 追加多段追击（20~35 段），每段 ×50%~60% |
| 白银风暴 | 任意部位被击碎 | 持续 15~20s，每次点击追加本次伤害 ×800%~1200% 的银风斩击 |
| 灭绝穿甲（崩塌） | 重甲部位点击 80~100 次 | 持续 8s，该部位护甲归零 + 全伤害放大 |
| 自动打击 | 重甲部位 8~15 次点击 | 自动发动 ×400%~800% 攻击力的碎甲重击 |
| 审判日 | 重甲部位点击 30~60 次 | 削除 Boss 最大 HP 的 60%~80% |
| 剥皮 | 暴击时概率 50%~80% | 将非弱点部位临时视为弱点，持续 8~15s |
| 末日审判 | 部位被击碎且 ≤30% HP | 标记 2~15 个随机部位，击碎标记部位获 25~40 死兆 |
| 致命出血 | 暴击时 | 4s 内 ×100%~300% 基础伤害 |
| 终末血斩 | 暴击 40~80 次 | 削除当前 HP 的 15%~25%，CD 30s |
| 死亡狂喜 | 死兆 ≥100 层 | 消耗 100 层，每层 ×100%~300% 基础伤害 |
| 死兆收割 | 死兆 ≥30 层（被动） | 30/60/90/120 层档位 → ×1.5/2.0/2.5/3.0 增伤 |
| 斩杀预兆 | 部位 HP ≤50%~65%（被动） | 每层死兆 +2%~5% 增伤 |
