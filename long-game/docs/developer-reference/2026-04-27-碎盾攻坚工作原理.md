# 碎盾攻坚（armor）工作原理

> 状态：当前有效。
>
> 本文只描述 2026-04-28 天赋重构后的现实现。完整数值与经济测算请以以下文档为准：
>
> - `docs/superpowers/specs/2026-04-28-talent-system-rework-design.md`
> - `docs/reports/2026-04-28-天赋成本调整总结.md`

## 核心机制

碎盾攻坚围绕“常驻破甲 -> 重甲累计命中 -> 触发崩塌 -> 崩塌状态增伤 -> 审判日部位削除”展开。

当前后端口径：

- `armor_core` 提供常驻护甲穿透与重甲崩塌触发
- 崩塌触发次数会随基石等级下降
- Tier1 层满奖励会继续额外减少崩塌触发次数
- `armor_collapse_ext`、`armor_ruin`、`armor_pen_convert`、`armor_ultimate` 均按等级成长

## 当前节点口径

| ID | 名称 | 层级 | 当前效果摘要 |
|---|---|---|---|
| `armor_core` | 灭绝穿甲 | Tier0 基石 | 常驻破甲，重甲累计命中触发崩塌；破甲率与触发次数随等级成长 |

| `armor_heavy_scale` | 以强制强 | Tier1 | 每 100 护甲增伤成长 |
| `armor_heavy_atk` | 重甲特攻 | Tier2 | 对重甲伤害成长型加成 |
| `armor_collapse_ext` | 崩塌延长 | Tier2 | 崩塌持续时间成长 |
| `armor_auto_strike` | 自动打击 | Tier2 | 自动打击间隔缩短、倍率提升 |

| `armor_ultimate` | 审判日 | Tier4 | 重甲累计命中后削除部位最大生命，触发次数与削除比例成长 |

## 层满奖励

| 层级 | 当前奖励 |
|---|---|
| Tier0 | 全伤害 +10% |
| Tier1 | 崩塌触发 -30，且全伤害 +10% |
| Tier2 | 护甲穿透 +15% |
| Tier3 | 崩塌易伤 +15% |
| Tier4 | 审判日削除 +10% |

## 成本口径

碎盾攻坚与其他两系共用统一成本模型，不再使用旧的固定成本 `100 / 2000 / 8000 / 30000 / 120000`。

```text
cost(level) = round(baseCost × level^0.85 × 1.8)
```

基准成本：

- Tier0 基石：`20`
- Tier1 主节点：`30`
- Tier2 主节点：`80`
- Tier3 主节点：`150`
- Tier4 终极：`200`
- Tier1 小节点：`15`
- Tier2 小节点：`35`
- Tier3 小节点：`60`

## 已废弃口径

以下说法均已废弃：

- “40% 破甲 + 100 次崩塌”的固定基石口径
- “审判日 100 次削 50%”的固定终极口径
- “首领猎杀只对 Boss 生效”的旧文案
- 固定成本 `100 / 2000 / 8000 / 30000 / 120000`
