# 版本更新公告 — 2026年5月

## 全新任务系统

任务系统正式上线！完成指定目标即可领取丰厚奖励。

- **前台任务页签**：主界面新增独立任务入口，随时查看进度
- **新增每日/每周/限时任务**：三种周期类型，限时任务设有截止时间
- **奖励多样化**：金币、灵石、天赋点、装备，应有尽有

---

## 天赋战斗 

**⚔️ 均衡攻势 —— 优势区间：均衡**
普攻连击便可持续触发多段追击与范围爆发，无需特定目标类型，应对任何敌人都能稳定输出高额伤害，适应性最强。

**🛡️ 碎盾攻坚 —— 优势区间：破甲**
专注攻击重甲部位可致其护甲崩塌，触发易伤与百分比削血，面对高护甲、高血量的重装型敌人拥有绝对压制力。

**🗡️ 致命洞察 —— 优势区间：弱点**
暴击即可积累死兆层数，逐级获得巨额增伤，对弱点部位发动致命收割，爆发上限为三大流派之最。

## 战斗视觉全面升级

战斗界面的每一个像素都经过了精心打磨。

- **伤害数字粒子爆炸**：不同伤害类型拥有差异化粒子特效，命中瞬间绽放
- **部位标签发光**：弱点（绿）、软组织（灰）、重甲（红），三种标签带发光效果，一眼识别
- **光标与命中闪光**：鼠标悬停格子有暗色光标，点击瞬间闪光 + 震屏
- **剑火花特效**：挥剑时粒子随行，连击渐变色越叠越炫
- **崩塌倒计时面板**：崩塌期间显示部位名 + 实时倒计时进度条
- **十字裁决图标**：右上角小图标标记，清晰不遮挡
