# 第一期：天赋特效清单

> 本期共 10 个特效，对应天赋系统的视觉表现。所有图片的像素规范参见 `通用像素约束.md`。
> JSON spec 文件位于 `pixel-assets/specs/`，渲染管线见 `pixel-assets/scripts/`。

## 厚重可读改造（全局覆盖）

> 从本次改造开始，本文旧的“低像素上限”仅作历史记录，新生成统一按“厚重可见”规则执行。  
> 目标：宁可少一些细节，也要在战斗里看得清、看得出体块。

| 类型 | 旧建议 | 新建议（厚重版） |
|------|--------|------------------|
| 小特效（血滴/小标记） | ≤10 | 14~24 |
| 图标类（节点/准星/徽标） | 10~15 | 18~32 |
| 中型特效（裂纹/冲击） | ≤16 | 24~42 |
| 大型特效（武器/序列帧单帧） | ≤25 | 36~72 |

额外要求：

- 主体最窄处不少于 2px。
- 至少两层明暗（高亮 + 暗边）。
- 主体包围盒至少占短边 45%。
