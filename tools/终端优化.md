### 一键安装

```bash
curl https://raw.githubusercontent.com/WwhdsOne/LearningNotes/master/tools/install_zsh.sh | bash
```

> 执行前请确保机器有curl

### 1. **安装 Zsh**

* **功能**：`Zsh` 是一个强大的 shell，作为默认的终端交互式 shell，提供比 Bash 更强的功能，比如自动补全、插件支持、和高度自定义能力。
* **安装方式**：

  * 对于 **macOS**，使用 **Homebrew** 安装。
  * 对于 **Ubuntu** 或其他基于 Debian 的系统，使用 `apt` 安装。
* **作用**：将 Zsh 设置为默认的 shell，以增强终端的交互体验。

```bash
brew install zsh
# 或者
sudo apt install zsh -y
```

**备注**：通过脚本中的 `chsh -s $(which zsh)`，将 `zsh` 设置为默认 shell。

### 2. **安装 Oh My Zsh**

* **功能**：**Oh My Zsh** 是一个开源的 Zsh 配置管理工具，提供了许多插件、主题、和强大的自定义选项。
* **安装方式**：

  * 使用官方提供的脚本，通过 curl 安装 **Oh My Zsh**。
* **作用**：简化 Zsh 配置，提供一个易于管理和扩展的 Zsh 环境。

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

### 3. **安装 Oh My Zsh 插件**

* **功能**：本脚本自动安装并启用以下插件：

  * **`zsh-autosuggestions`**：为 Zsh 提供命令自动补全建议。
  * **`zsh-syntax-highlighting`**：为 Zsh 提供命令语法高亮功能，增强代码可读性。
* **安装方式**：通过 Git 克隆插件到 **Oh My Zsh** 的插件目录。
* **作用**：增强 Zsh 使用体验，提升命令行操作效率。

```bash
git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
```

### 4. **安装 Starship**

* **功能**：**Starship** 是一个现代化的命令行提示符，支持跨平台且高度可定制，能够显示 Git 状态、当前目录信息、CPU/内存使用情况等。
* **安装方式**：

  * 在 **macOS** 使用 Homebrew 安装。
  * 在 **Ubuntu** 使用 `curl` 安装。
* **作用**：为终端提供一个美观且功能强大的提示符，使得信息更加清晰，提升终端使用体验。

```bash
brew install starship
# 或者
curl -sS https://starship.rs/install.sh | sh
```

**备注**：`eval "$(starship init zsh)"` 被添加到 `~/.zshrc` 文件中，确保在启动时初始化 Starship。

### 5. **安装 Nerd Font（Hack Nerd Font）**

* **功能**：**Hack Nerd Font** 是一种为开发者设计的字体，支持包括 Powerline 和 Unicode 字符在内的符号，非常适合用于命令行工具和代码编辑器中。
* **安装方式**：

  * 在 **macOS** 使用 Homebrew 安装。
  * 在 **Ubuntu** 需要手动安装字体（脚本中提醒用户手动安装）。
* **作用**：提高终端和编辑器的可视化效果，特别是对于 **Starship** 提示符等需要符号的环境。

```bash
brew tap homebrew/cask-fonts
brew install --cask font-hack-nerd-font
```

### 6. **配置 Catppuccin Powerline 主题**

* **功能**：**Catppuccin** 是一个美观且高对比度的主题风格，适合用于终端和代码编辑器。通过 **Starship** 配置该主题，提供一个漂亮的命令行提示符。
* **安装方式**：使用 Starship 提供的预设命令来配置 **Catppuccin Powerline** 主题，并将其写入 `~/.config/starship.toml`。
* **作用**：让终端界面更具美感，配合 **Starship** 使用时，显示更为优雅的提示信息。

```bash
starship preset catppuccin-powerline -o ~/.config/starship.toml
```

### 7. **安装工具：bat、glow、poppler-utils、eza、hexyl、mediainfo、exiftool**

* **功能**：这些工具分别用于：

  * **`bat`**：带有语法高亮的 `cat` 替代命令，适合查看代码文件。
    * `bat`工具用的时候前缀其实是`batcat`，直接输入`bat`会出现`command not found`
  * **`glow`**：命令行 Markdown 渲染器，直接在终端中渲染 Markdown 文件。
  * **`poppler-utils`**：PDF 工具集，包含 `pdftotext` 等工具，用于处理 PDF 文件。
  * **`eza`**：`ls` 的增强版本，提供颜色、排序和图形化支持。
  * **`hexyl`**：十六进制文件查看器。
  * **`mediainfo`**：查看多媒体文件的详细信息。
  * **`exiftool`**：查看和修改文件的 EXIF 信息，通常用于图片和音频文件。
* **安装方式**：通过 **Homebrew** 安装（**macOS**）或使用 `apt` 安装（**Ubuntu**）。
* **作用**：提高文件查看和处理效率，特别是在命令行工作时。

```bash
brew install bat glow poppler-utils eza hexyl mediainfo exiftool
# 或者
sudo apt install -y bat glow poppler-utils eza hexyl mediainfo exiftool
```

### 8. **创建 `yazi.toml` 配置文件**

* **功能**：为 **yazi** 创建并配置 `yazi.toml` 文件，设定窗口布局、文件排序方式、隐藏文件显示、以及图像预览等功能。
* **配置内容**：

  * **`ratio`**：定义父目录、当前目录和预览窗口的比例。
  * **`sort_by`**：指定文件排序方式（如按字母、大小、创建时间等）。
  * **`show_hidden`**：是否显示隐藏文件。
  * **`image`**：启用图像预览功能，支持图片显示和调整图像质量。
* **作用**：自定义 **yazi** 行为，支持图片预览、文件排序、隐藏文件管理等功能。

配置内容：

```toml
[mgr]
ratio = [1, 2, 8]
sort_by = "alphabetical"
sort_dir_first = true
show_hidden = false

[preview]
image = true
max_width = 900
max_height = 1350
image_quality = 80
image_filter = "triangle"
```

### 9. **生效配置**

* **功能**：通过 `source ~/.zshrc` 命令确保所有 Zsh 配置生效，使 **Starship** 和 **Oh My Zsh** 插件能够在终端中生效。

```bash
source ~/.zshrc
```
