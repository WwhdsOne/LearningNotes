# 查看当前运行内存使用占全部运行内存比率

```shell
free | awk 'FNR == 2 {print $3/($3+$4)*100}'
```

这个命令首先使用`free`命令获取内存使用情况，然后使用`awk`命令处理输出结果。

`FNR == 2`表示处理第二行（即内存的行），`$3/($3+$4)*100`表示计算已使用内存（$3）占总可用内存（已使用内存+$4）的比率，并将结果乘以100转换为百分比。
