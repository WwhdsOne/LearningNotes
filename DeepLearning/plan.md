## 🎯 附录：M1 Air 优化备忘录

### 8GB 内存专属配置
```python
# config.yaml 或 config.py
BATCH_SIZE = 8  # 不是32
NUM_WORKERS = 2  # 不是4或8
PIN_MEMORY = False  # MPS 下不需要
GRAD_ACCUMULATION = 4  # 等效 batch = 32
MIXED_PRECISION = True  # 必须开
```
